# Landbank API

API for Landbank.

## Prerequisites

* Node JS version 10 or above.

## Installation

1. Install node modules after the repository is cloned.

```
    $ npm install
```

## Run the Application

1. Start the development server

```
    $ npm start
```

- http://localhost:3000
- press Ctrl + C to stop the development server.
